设置 json 对象到 表单元素中, 映射规则按照 json对象的属性名 追加前缀进行对应

参数说明：

  • s json 对象
  • prefix 映射到 表单域中的前缀字符串 ,可为空
  • splitfix 如果json 中有 多选等字段, 该值自动转换成数组的 分隔符，默认为 , 号
  • parentId 指定只更新某个id 以内的数据

参数说明：

  • s “555.555”或者555.555
  • n 保留几位，0<=n<=20，非此范围n=2；n==null则 n=2
